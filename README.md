# PCA Face Recognition on ORL Dataset

This project implements PCA (Principal Component Analysis) for dimensionality reduction on the ORL face dataset, followed by classification using various classifiers.

## Project Structure
- `orl_faces/`: ORL face dataset (s1-s40 folders)
- `src/`: Source code for PCA and classification
- `results/`: Results and visualizations
